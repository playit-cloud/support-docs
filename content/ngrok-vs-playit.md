### Ngrok vs Playit.gg

## Using Ngrok

Ngrok is a service that works very similarly to playit.gg. You download the agent, run it, and you can configure tunnels. The downside to Ngrok is that it often can give you bad routing, and it is also hard to reach support (if any). Ngrok is also not optimised for gaming, which can lead to lag spikes or disconnects.

## Using playit.gg

Playit.gg is a service origionally built for gaming. This leads to Playit.gg being more stable, and lower latency then other services like Ngrok. Playit.gg provides [static IP addresses](https://whatismyipaddress.com/dynamic-static) so players never have to type in a new IP ever again. Playit.gg also has a [Discord server](https://discord.gg/wUdKqGjuTb) where you can reach the developers and amazing community support. You can also request routing changes if you experience high latency with your friends.
